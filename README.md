## Library Management API  
**Backend - Spring Boot 3 + Java 17**

REST API quản lý thư viện: sách, mượn trả, phân quyền.

## Tech

- Java 17, Spring Boot 3  
- Đăng nhập bằng session 
- JPA/Hibernate, SQL Server  
- Maven

## Chạy

./mvnw spring-boot:run
